# My Django Shop 🛒

Простой каталог товаров на Django с использованием Class-Based Views.

## Как запустить

1. **Клонировать репозиторий:**
   `git clone <ссылка>`

2. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt

3. **Сделать миграции:**
```bash
python3 manage.py makemigrations products
```

4. **Запустить сервер:**
```bash
python3 manage.py runserver
